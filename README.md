# Usage
<!-- usage -->
```sh-session
$ npm install -g gitlab-commander
$ gitlab-commander COMMAND
running command...
$ gitlab-commander (-v|--version|version)
gitlab-commander/0.0.1 linux-x64 node-v14.15.5
$ gitlab-commander --help [COMMAND]
USAGE
  $ gitlab-commander COMMAND
...
```
